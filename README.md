# totess studio Website

A minimalist, professional website for totess studio - a software and gaming startup.

## Features

- Clean, minimalist design
- Fully responsive layout
- Smooth scrolling navigation
- Fade-in animations on scroll
- Modern typography using Inter font
- Single-page application structure

## Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Navigation and scroll animations
- `README.md` - This file

## Setup

1. Open `index.html` in a web browser
2. No build process or dependencies required - pure HTML, CSS, and JavaScript

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --color-primary: #000000;
    --color-secondary: #666666;
    --color-accent: #333333;
    --color-bg: #ffffff;
    --color-bg-alt: #fafafa;
}
```

### Content
Update the content directly in `index.html`:
- Hero section
- About section
- Projects section
- Contact information

### App Store Link
Update the App Store link in the projects section with your actual App Store URL.

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile responsive design
